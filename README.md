## Installation

1. Clone the repository:

```bash
  https://github.com/Teekay1702/Book-App.git
```
    
2. Navigate to the project folder:

```bash
  cd book-app
```

3. Install dependencies:

```bash
  npm install
  npm install react-router-dom
```

4. Start the development server:

```bash
  npm run dev
```
